◇ 概要 ◇
　郵政省の公開している12万件の郵便番号データを使ってAccessの住所入力を効率化します。

◇ 動作条件 ◇
　Access 2003 以降 のバージョン

◇ インストール ◇
　zip ファイルを解凍して 中にある mdb ファイルをマイドキュメント等の
　フォルダにコピーしてください。

◇ つかいかた ◇
  ●最初に
　　住所入力.mdb ファイルをダブルクリックして郵便番号を入力してください。
　　自動的に住所が入力されます。

  ●カスタマイズ
　　ある程度 Access で開発できるノウハウをお持ちの方は
　　カスタマイズ方法.txt を見てプログラムをカスタマイズすることができます。
